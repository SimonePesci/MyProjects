Per poter avviare correttamente il codice è necessario utilizzare una versione di Java 11.0 o superiore.

Per avviare correttamente il codice è necessario:
0 Modificare se si vuole il file prova.txt in Resources per codificare e decodificare un testo a scelta
1 Aprire il prompt dei comandi
2 Posizionarsi nella cartella "Huffman" 
3 Scrivere: java -jar Huffman.jar e dare invio.
